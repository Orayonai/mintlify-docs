---
title: 'Gerenciar Lembretes'
api: 'POST /rest/v1/rpc/block_lead_reminders_by_phone'
description: 'Bloqueia ou desbloqueia disparos de lembrete para um lead'
---

## Objetivo

Controlar se um lead deve receber lembretes automaticos (disparos em massa). Util para:
- Lead que pediu para nao receber mensagens
- Lead que ja confirmou presenca
- Controle de spam/frequencia

---

## Bloquear Lembretes

### Endpoint

```
POST {SUPABASE_URL}/rest/v1/rpc/block_lead_reminders_by_phone
```

### Request Body

<ParamField body="p_phone_number" type="string" required>
  Telefone do lead
</ParamField>

```json
{
  "p_phone_number": "+5511999999999"
}
```

### Response

```json
{
  "success": true,
  "message": "Lembretes bloqueados"
}
```

### Exemplo cURL

```bash
curl -X POST '{SUPABASE_URL}/rest/v1/rpc/block_lead_reminders_by_phone' \
  -H 'apikey: SUA_ANON_KEY' \
  -H 'Authorization: Bearer SUA_SERVICE_KEY' \
  -H 'Content-Type: application/json' \
  -d '{"p_phone_number": "+5511999999999"}'
```

---

## Desbloquear Lembretes

### Endpoint

```
POST {SUPABASE_URL}/rest/v1/rpc/unblock_lead_reminders_by_phone
```

### Request Body

```json
{
  "p_phone_number": "+5511999999999"
}
```

### Response

```json
{
  "success": true,
  "message": "Lembretes desbloqueados"
}
```

### Exemplo cURL

```bash
curl -X POST '{SUPABASE_URL}/rest/v1/rpc/unblock_lead_reminders_by_phone' \
  -H 'apikey: SUA_ANON_KEY' \
  -H 'Authorization: Bearer SUA_SERVICE_KEY' \
  -H 'Content-Type: application/json' \
  -d '{"p_phone_number": "+5511999999999"}'
```

## Exemplo N8N - Bloquear

```json
{
  "name": "BloquearDisparos",
  "method": "POST",
  "url": "{SUPABASE_URL}/rest/v1/rpc/block_lead_reminders_by_phone",
  "body": {
    "p_phone_number": "={{ $json.body.messages[0].conversation.contact_inbox.source_id }}"
  }
}
```

## Exemplo N8N - Desbloquear

```json
{
  "name": "DesbloquearDisparos",
  "method": "POST",
  "url": "{SUPABASE_URL}/rest/v1/rpc/unblock_lead_reminders_by_phone",
  "body": {
    "p_phone_number": "={{ $json.body.messages[0].conversation.contact_inbox.source_id }}"
  }
}
```

## Casos de Uso

| Acao | Funcao |
|------|--------|
| Lead pediu para parar de receber | `block_lead_reminders_by_phone` |
| Lead confirmou presenca | `block_lead_reminders_by_phone` |
| Lead quer voltar a receber | `unblock_lead_reminders_by_phone` |
| Novo evento, reativar todos | `unblock_lead_reminders_by_phone` |

<Note>
  O bloqueio afeta apenas disparos em massa (campanhas). Mensagens individuais da IA continuam funcionando.
</Note>
