## Arquitetura

```
┌──────────────────────────────────────────────────────────────────────────────┐
│                              VISAO GERAL DO SISTEMA                          │
└──────────────────────────────────────────────────────────────────────────────┘

  ┌─────────────────┐      ┌─────────────────┐      ┌─────────────────┐
  │    WhatsApp     │      │      N8N        │      │    Supabase     │
  │    Business     │─────▶│    Workflow     │─────▶│  Edge Function  │
  │  META / UAZAPI  │      │                 │      │                 │
  └─────────────────┘      └─────────────────┘      └────────┬────────┘
                                                             │
                           ┌─────────────────────────────────┼─────────────────┐
                           │                                 ▼                 │
                           │  ┌──────────────────────────────────────────┐    │
                           │  │           PROCESSAMENTO (Steps)          │    │
                           │  │                                          │    │
                           │  │  1. Organization  ──▶  organizations     │    │
                           │  │  2. Contact       ──▶  contacts          │    │
                           │  │  3. Session       ──▶  sessions          │    │
                           │  │  4. Message       ──▶  messages          │    │
                           │  │  5. Buffer (Redis)                       │    │
                           │  │  6. Whisper (audio)                      │    │
                           │  │  7. Vision (imagens)                     │    │
                           │  └──────────────────────────────────────────┘    │
                           │                                                   │
                           └───────────────────────────────────────────────────┘
                                                             │
                                                             ▼
  ┌─────────────────┐      ┌─────────────────┐      ┌─────────────────┐
  │     OpenAI      │      │   Redis/Upstash │      │   LangChain     │
  │  Whisper+Vision │◀────▶│     Buffer      │◀────▶│   Agent + RPC   │
  └─────────────────┘      └─────────────────┘      └─────────────────┘
```

### Componentes Principais

| Camada | Componente | Funcao |
|--------|------------|--------|
| **Entrada** | META Cloud API | API oficial WhatsApp Business |
| **Entrada** | UAZAPI | API alternativa WhatsApp |
| **Orquestracao** | N8N Workflow | Recebe webhooks, normaliza dados |
| **Processamento** | Edge Function | Processa mensagens em steps |
| **Storage** | PostgreSQL | Armazena organizations, contacts, sessions, messages |
| **Cache** | Redis/Upstash | Buffer de mensagens, memoria do agente |
| **IA** | OpenAI | Whisper (audio) + Vision (imagens) |
| **Agente** | LangChain | Agente IA com tools/RPCs |

## Fluxo de Mensagem

<Steps>
  <Step title="Recebimento">
    WhatsApp envia webhook para N8N (META Cloud API ou UAZAPI)
  </Step>
  <Step title="Normalizacao">
    N8N extrai e normaliza dados no formato padrao
  </Step>
  <Step title="Processamento">
    Edge Function processa: Organization → Contact → Session → Message
  </Step>
  <Step title="Buffer">
    Redis agrupa mensagens consecutivas (7 segundos)
  </Step>
  <Step title="Transcricao">
    OpenAI Whisper transcreve audios para texto
  </Step>
  <Step title="Agente IA">
    LangChain processa e gera resposta inteligente
  </Step>
  <Step title="Resposta">
    Mensagem enviada de volta ao WhatsApp
  </Step>
</Steps>
